# Music21 Music Project

This project uses music21 for creating and playing music in Python.

## Setup

1. **Virtual Environment**: Already created at `venv/`
2. **Dependencies**: music21 is installed in the venv

## Running the Music

```bash
source venv/bin/activate
python test.py
```

The script will open a MIDI player to play the music.

## Files

- `test.py` - Main music code with piano and bass parts
